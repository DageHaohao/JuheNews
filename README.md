# JuheNews
<font size=4>根据聚合数据API提供的免费接口实现一个自己的应用，主要采用了RxAndroid响应式编程，和Material Design库中的一些内容，同时添加了部分开源库，实现一些基本简单的效果，希望各位指正交流！</br></br>
目前 有4个模块：</br></br>
1. 新闻头条</br></br>
2. 笑话大全</br></br>
3. 趣味图片</br></br>
4. 历史上的今天</br></br></font>
